[FR] Texture Pack :
--------------

Décompressez cette archive comme vous l'avez fait pour installer Power Auras, vous devriez avoir davantage de textures dans le dossier de l'AddOn. Lancez le jeu, tapez /powa, allez dans les Options Avancées et réglez le paramètre "Nombre maximum de textures" afin qu'il coincide avec le nombre du dernier fichier installé (si vous installez le "TexturePack21-30", vous aurez donc un maximum de 30 textures).


----------------------------------------------------------------------------------------------------------------------------


[EN] Texture Pack :
--------------

Unzip this file like you do with the Mod, you will now have some more textures on the Power Auras directory. Just run the game, type /powa and go to Advanced Options to put the parameter "Max number of textures" to the last installed texture file (if you have installed the "TexturePack21-30", you have now a maximum of 30 textures).
